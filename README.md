# Tarea 5 - Métodos de ordenamiento

## **📘 Descripción general**
Este proyecto consiste en desarrollar un programa **modular** en Python que implemente los algoritmos de ordenamiento vistos en clase:
- Bubble Sort
- Insertion Sort  
- Selection Sort  
- Shell Sort  
- Merge Sort  
- QuickSort (solo pivote central)

El programa deberá:
1. Leer un archivo `nombres.txt` con 50 nombres.  
2. Guardar los nombres en una **lista de strings**.  
3. Mostrar un **menú interactivo** para elegir el algoritmo de ordenamiento.  
4. Permitir ordenar en **ascendente o descendente** (ascendente por omisión).  
5. Mostrar la lista ordenada.  
6. Estar dividido en **dos módulos**:

```
main.py
ordenamientos.py
nombres.txt
```
